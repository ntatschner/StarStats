# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its
code of conduct. The full canonical text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Scope

The code of conduct applies to all project spaces — the GitHub
repository (issues, pull requests, discussions, code review),
official communication channels, and any in-person or online events
held under the project's name.

It also applies when an individual is officially representing the
project in public spaces — for example, posting from an official
account, acting as an appointed representative at an event, or
publishing project-related content on personal channels in a way
that a reasonable observer would associate with StarStats.

## Reporting

If you experience or witness behaviour that violates the code of
conduct, please report it to **conduct@starstats.app**.

Reports will be:

- Acknowledged within 7 days.
- Reviewed by the project maintainer.
- Kept confidential to the extent compatible with investigating
  the report and (where applicable) any subsequent enforcement
  action.

If the report concerns the maintainer themselves, escalate via the
contact channels listed at <https://www.contributor-covenant.org/faq/>.

## Enforcement

Maintainers who do not follow or enforce the code of conduct in
good faith may face temporary or permanent repercussions as
determined by the project's governance.

The maintainer will use the enforcement guidelines published with
Contributor Covenant 2.1 to determine the consequences of any
action they deem in violation:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines>
